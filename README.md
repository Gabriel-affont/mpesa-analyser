# M-Pesa Statement Analyser

A web app that parses your M-Pesa CSV statement and lets you filter
transactions by contact — see exactly how much you've sent and received
with any person.

## Features
- Upload CSV statement (stays in browser, never sent to a server)
- Search contacts by name or phone number
- Per-contact breakdown: total sent, received, and net balance
- Filter transactions by direction (sent / received)

## Tech stack
- React (Vite)
- Vanilla CSS

## Run locally
npm install && npm run dev
